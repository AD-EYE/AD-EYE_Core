
% ================================================================================================================
% Follow the next steps to create your own experiment with the Demo_TISDriftAndNoise
% ================================================================================================================

Step 1.	Open the Prescan GUI and open the Demo_TISDriftAndNoise.pex file. 

Step 2. click on the Invoke Simulation Run Mode Button and Matlab will be opened.

Step 3. Check if Matlab open the directory where the experiment was saved. Then open the simulink file
	Demo_TISDriftAndNoise_cs.slx file using Matlab.

Step 4. Start the Simulink model and wait until the simulation is completely finished.

Step 5. Run the script PlotHistogram.m (e.g. "PlotHistogram('simout.mat')") at the Matlab command line.
