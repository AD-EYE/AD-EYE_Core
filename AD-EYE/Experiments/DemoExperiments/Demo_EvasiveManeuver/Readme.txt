
% ================================================================================================
% Follow the next steps to create your own experiment with the Demo_EvasiveManeuver
% ================================================================================================

Step 1.	Open the Prescan GUI and open the Demo_EvasiveManeuver.pex file.

Step 2. click on the Invoke Simulation Run Mode Button and Matlab will be opened.

Step 3. In Matlab go to the directory where the experiment was saved (../Demo_EvasiveManeuver). Then open the simulink file Demo_EvasiveManeuver_cs.slx file using Matlab.

Step 4. Start the Simulink model. (An initialisation function is implemented in order to load the model parameters)

Step 5. See the 3D visualisation viewer for the simulation result.
